¿Qué es DevOps Hub?

DevOps Hub es una plataforma web educativa diseñada para simular entornos profesionales de gestión de proyectos al estilo de herramientas como Jira, Trello o Azure DevOps.

Su objetivo es permitir a los usuarios gestionar proyectos, tareas y equipos de trabajo en un entorno controlado donde se ponen en práctica conceptos de desarrollo ágil, integración continua y trabajo colaborativo.

Este proyecto ha sido desarrollado como ejercicio final del ciclo de Desarrollo de Aplicaciones Web, y busca integrar conocimientos de frontend, backend y despliegue en la nube de forma práctica y cohesionada.

Tecnologías utilizadas:

  • Frontend: HTML, CSS, Bootstrap 5 y JavaScript
  • Backend: Node.js con Express
  • Base de datos: SQLite (modo local)
  • Despliegue: Plataforma Render y versión alternativa dockerizada
  • Control de versiones: Git + GitHub
